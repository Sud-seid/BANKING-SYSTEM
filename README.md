# Enhanced Banking System

A simple banking system application written in Java.

## How to Run

1. Compile the Java files:
   ```bash
   javac EnhancedBankingSystem.java
   ```
2. Run the application:
   ```bash
   java EnhancedBankingSystem
   ```

## Requirements
- JDK 11 or higher
- JavaFX (if applicable)
